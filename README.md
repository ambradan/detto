## What it does

- Records audio with noise suppression
- Transcribes and cleans the text (removes non-verbal sounds, hesitations)
- Allows full manual editing before use
- Converts final text back to audio
- Supports long pauses without auto-stopping
- Never invents content â€” prefers omission over fabrication

---

## Design principles

- User controls the final output, always
- Silence and pauses are normal, not errors
- Incomplete output is better than incorrect output
- AI assists, it does not author
- Preserve the user's voice and tone, do not flatten personality
