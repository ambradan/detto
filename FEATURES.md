# Core features

**Input**
- Audio recording with tap-to-start / tap-to-stop
- Noise suppression, echo cancellation, auto gain control
- Optional text input as alternative to voice
- Optional context field (voice or text) to guide interpretation

**Processing**
- Transcription with non-verbal sound removal
- Medical device noise filtering (ventilator, respirator)
- No auto-stop on silence or long pauses
- Unclear content omitted or marked, never invented

**Output**
- Full manual editing of generated text
- Format selection: WhatsApp, Email, LinkedIn
- Style selection: personal tone or neutral
- "Rewrite clearer" option
- Undo / Retry actions

**Playback**
- Text-to-speech with auto-play on first listen
- Replay available via player controls

**Quick phrases**
- Pre-defined phrases for common situations
- One-tap copy and insert
- User can add custom phrases

All features minimize effort and respect user agency.
